<h1>Electronic Shopping</h1>

<p>
Electronic Shopping is a frontend web application built using Vite, HTML, CSS, and JavaScript.
It provides a simple electronic store interface with product listings and basic client-side interactions.
</p>

<h2>Features</h2>
<ul>
  <li>Multi-page electronic store layout</li>
  <li>Product listings with images</li>
  <li>Responsive user interface</li>
  <li>Client-side functionality using JavaScript</li>
</ul>

<h2>How to Run</h2>

<p>Requirements: Node.js (v16 or higher) and npm.</p>

<pre><code>git clone https://github.com/hritikio/Electronic-Shopping.git
cd Electronic-Shopping
npm install
npm run dev</code></pre>

<p>Open the application in your browser:</p>

<pre><code>http://localhost:5173</code></pre>
